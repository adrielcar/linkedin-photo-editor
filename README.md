LinkedIn Photo Editor

Aplicação web para transformar fotos comuns em fotos de perfil profissional.

Funcionalidades

- Recorte automático da imagem
- Centralização
- Estilo profissional
- Download da imagem

Tecnologias utilizadas

- JavaScript
- HTML
- CSS
- Canvas API

Como usar

1. Faça upload de uma imagem
2. A imagem será processada automaticamente
3. Clique em "Baixar Foto"

Objetivo

Projeto desenvolvido para praticar manipulação de imagens com JavaScript e conceitos de assincronicidade.
